# 🚀 Getting Started with AI Learning Hub

## Prerequisites

- Python 3.8 or higher
- Git
- Basic command line knowledge
- Text editor or IDE (VS Code, PyCharm, Jupyter)

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/AnushkaMunde074/AI-Learning-Hub.git
cd AI-Learning-Hub
```

### 2. Create Virtual Environment

```bash
# macOS/Linux
python3 -m venv venv
source venv/bin/activate

# Windows
python -m venv venv
venv\Scripts\activate
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

1. Install Python extension
2. Install Jupyter extension
3. Select interpreter from `venv`

### PyCharm
1. Open project
2. Configure interpreter from `venv`
3. Run Python scripts directly

## Troubleshooting

### Python Version Issues
```bash
python3 --version  # Should be 3.8+
```

### Package Installation Issues
```bash
# Upgrade pip
pip install --upgrade pip

# Install specific version
pip install scikit-learn==1.0.2
```

### Jupyter Not Found
```bash
pip install jupyter
jupyter --version
```
